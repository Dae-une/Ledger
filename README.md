# 가계부 및 일정 관리 앱


# 주요기능
### 1. 로그인
*  구글을 이용한 소셜 로그인
### 2. 가계부 CRUD
*  날짜를 선택하여 소비내역을 기록하고, 날짜마다 소비내역을 볼 수 있습니다.
*  또한, 프로필 페이지에서 한달간의 소비내역을 파이차트를 이용하여 확일할 수 있습니다.<br/>
